# 22 Years of Surksha - Setup Instructions

Welcome to your personalized birthday website for Surksha!

## 1. Adding Photos
The gallery expects 8 photos named `1.jpg`, `2.jpg`, ... up to `8.jpg`.
Please place your memory photos in the folder:
`public/memories/`

If you don't add them, the site will show placeholder images.

## 2. Adding Music
The site plays background music.
Please place your music file named `music.mp3` in the `public/` folder.
Location: `public/music.mp3`.

## 3. Running the Website
To preview the site on your computer:
1. Open a terminal in this folder.
2. Run `npm run dev`.
3. Open the link shown (usually http://localhost:5173).

## 4. Deploying (Pushing to Git)
This project is ready to be pushed to GitHub.
If you haven't already:
1. Create a repository on GitHub.
2. Run:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/suru-bday.git
   git branch -M main
   git push -u origin main
   ```

## 5. Hosting
The easiest way to host is using Vercel or Netlify.
1. Go to Vercel.com and login.
2. Import your GitHub repository.
3. Click Deploy. Markdown magic!

Enjoy your celebration! ❤️
